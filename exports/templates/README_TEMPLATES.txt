# Plantillas de Archivos para Copiloto de Supervivencia Financiera

Este directorio contiene plantillas CSV de ejemplo para los diferentes tipos de archivos que puedes subir a la aplicación.

## Archivos Disponibles

### 1. bank_statement_template.csv
Plantilla para extracto bancario (OBLIGATORIO)

### 2. invoices_sales_template.csv
Plantilla para facturas emitidas (pendientes de cobro)

### 3. invoices_purchase_template.csv
Plantilla para facturas recibidas (pendientes de pago)

### 4. presupuesto_vs_real_template.csv (futuro)
Para comparación presupuesto vs real

### 5. projects_template.csv (futuro)
Para seguimiento de proyectos

## Cómo Usar

1. Descarga la plantilla correspondiente
2. Rellena con tus datos (respeta el formato de columnas)
3. Guarda como CSV o Excel
4. Sube a la aplicación

## Notas Importantes

- Las fechas pueden estar en formato DD/MM/YYYY o YYYY-MM-DD
- Los importes deben usar punto (.) como separador decimal
- No uses comas en los importes (ej. 1500.00 no 1,500.00)
- Los nombres de columnas NO son sensibles a mayúsculas
- Puedes añadir columnas adicionales, serán ignoradas

## Soporte

Si tienes problemas con el formato, revisa la sección de Ayuda en la app o el README.md principal.
